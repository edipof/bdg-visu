Sobre o Trabalho

Proposta

O trabalho em questão desenvolve uma visualização para observar onde no território brasileiro os 12 principais clubes do Brasil são referênciados através de tweets geolocalizados feitos no ano de 2014. Com esta visualização podemos avaliar se realmente alguns times tem reconhecimento nacional e torcedores espalhado por todo o país.

Tratamento dos dados

Os dados utilizados foram coletados do Twitter através da API que ele disponibiliza. São dados com coordenadas geográficas de todo ano 2014. A base de dados inicial, com tweets, coordenadas geográficas e nome da cidade e estado onde o tweet foi realizado, possuia cerca de 25gb de dados no formato tsv. Os dados foram filtrados e tratados utilizando algumas técnologias que seram mostradas mais a frente no decorrer do relatório.

Para realizar o tratamento dos dados foi utilizado o Apache Spark que é um processador de dados em larga escala muito rápido, rodando em cima da estrutura do Hadoop. Para que o processamento fosse possível, foram desenvolvidos scripts para realizar a filtragem desses dados, estes foram desenvolvidos em Scala uma linguagem funcional advinda do Java que é uma das linguagens utilizadas para escrever scripts para rodar no Apache Spark.

Como a base de dados era muito grande e o tempo não era suficiente para fazer um trabalho minucioso para obter todos os dados referentes aos times avaliados, então optou­se pela utilização das principais hashtags (#) que os torcedores desses times usam para referência­los na web. As '#' utilizadas podem ser vistas em formato de nuvem aqui.

Estes dados, após serem filtradas, foram exportados para o formato GeoJson, através de um script feito em Ruby, para facilitar a leitura do dado no momento da visualização

Visualização

Para gerar a visualização dos dados foi utilizado uma biblioteca javascript chamada D3.js (Data Driven Documents) é usada para manipulação de documentos baseados em dados. Utilizando essa biblioteca é possível criar visualização de gráficos interativos usando HTML, CSS e SVG.

Por ser uma biblioteca javascript, tem foco em padrões web, podendo ter comportamentos diferentes nos diversos navegadores existentes. Essa biblioteca permite criar diversos tipos de gráficos e figuras geométricas.

A principio as visualizações foram implementadas utilizando como entrada os arquivos em formato Geojson gerado pelo script em Ruby, porem este é um formato de dados muito grande e estavam deixando as visualizações pesadas. Então como alternativa foram utilizados como entrada arquivos do tipo Topojson que é um dado tipo de dado com todas as informações de um Geojson porém de forma comprimida o que diminui consideravelmente o tamanho dos dados.

Avaliação da Visualização

Após a finalização da visualização, foi possivel tirar algumas conclusões em cima daquilo que implementado, por exemplo, foi possível observar que times como Corinthians e Flamengo tem um grande numero de torcedores espalhados pelo Pais, contendo tweets em todo Brasil, inclusive no Acré, Rondônia, entre outros estados. Uma possivel causa desse espalhamento, pode estar diretamente relacionado ao numero elevado de jogos transmitidos destes times para todo o Brasil pelas TVs abertas.

Um fato curioso que não era esperado foi o grande numero de tweets relacionados ao Atletico-MG espalhados por todo o pais, o esperado era que estes dados estivessem centralizados em Minas Gerais. Uma hipotese que pode estar relacionada a este fenômeno é o fato deste time vir de uma série de conquistas de titulos importantes nos ultimos anos, o que pode ter aumentado numero de seus torcedores em todo o terrtório nacional.

Outro fato curioso é que alguns times são citados também fora do Brasil, o Palmeiras é um excelente exemplo deste fato, que a principio não obtivemos nenhum possivel argumento que possa justificar essa situação.
